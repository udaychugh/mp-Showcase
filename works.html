Welcome to the guide for using TravelMate - your travel guide.

Sign up
  1. 1) New user has to register for using TraveMate by filling all the required details.
  2. 2) The user must enter the details of the vehicle used for the trip,like vehicle model,fuel type etc.
  3. 3) Existing user can login directly and use TravelMate.
Real Time Map
  1. 1) Enter your starting location and the destination of your trip.
  2. 2) Best routes would be shown having the real time traffic details.
Fuel Consumption
  1. 1) One can check the amount of fuel consumed in the journey.
  2. 2) User can locate nearby fuel stations in case of refill.
  3. 3) Information about the fuel price is shown.
Weather Information
  1. 1) User can check the weather conditions for a more comfortable journey.
  2. 2) User can see the weather details of the starting location and the destination of the trip.
Journey Analysis
  1. 1) Cost of the journey is shared with the user.
  2. 2) Maintenance expenses is also shared with the user with the help of interactive charts.
